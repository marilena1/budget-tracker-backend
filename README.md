# budget-tracker-backend
Spring Boot backend for the Budget Tracker App (CF8 final project)

## General Requirements
- Java 17 (tested with Amazon Corretto 17.0.14_7)
- Gradle 9.2.1

## Small Description about how the application runs via the CLI
- BUILD -> gradle clean build
- RUN -> java -jar ./build/libs/budget-tracker-backend-0.0.1-SNAPSHOT.jar

## Environment variables
- MONGODB_URI= 
- JWT_SECRET_KEY=
